Detailed Description

Nested Attribute Dictionary.

A class to convert a nested Dictionary into an object with key-values accessible using attribute notation (AttrDict.attribute) in addition to key notation (Dict["key"]). This class recursively sets Dicts to objects, allowing you to recurse into nested dicts (like: AttrDict.attr.attr)
