def btx.interfaces.ipsana.PsanaInterface.distribute_events (   self,
  rank,
  total_ranks,
  max_events = -1 
)

For parallel processing.

Update self.counter and self.max_events such that events will be distributed evenly across total_ranks, and each rank will only process its assigned events. Hack to avoid explicitly using MPI here.

Parameters

rank : int current rank total_ranks : int total number of ranks max_events : int, optional total number of images desired, option to override self.max_events

Retrieve a fixed number of images from the run.

If the pedestal or gain information is unavailable and unassembled images are requested, return uncalibrated images.

Parameters

Returns
