Calculate the residual for least square optimization.

The residual is computed as the difference between the intensity values of pixels that belong to the rings and the image's maximum intensity value.

Parameters

params : dict parameters for optimization img : numpy.ndarray a powder image kwargs : dict additional key-value arguments

Returns

pvals : numpy.ndarray pixel values subtracted by the max pixel value in the image
